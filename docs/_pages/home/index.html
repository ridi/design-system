RIDI Design System

리디를 디자인하기 위한 단 하나의 시스템.

  • Principles

    모두가 함께 고민하여 만든 원칙으로 우리는 더 효율적인 디자인 의사결정을 할 수 있습니다. 원칙은 제약이 아닙니다. 원칙으로 리디만의 일관된 사용 경험을 만들어 나갈 수 있습니다.

    더 알아보기

  • Writings

    잘 정의된 UI 텍스트는 자연스러운 사용자 경험의 뼈대가 됩니다. 리디의 제품에서 사용자가 보는 텍스트에는 우리만의 고민과 원칙이 녹아있습니다.

    더 알아보기

  • Colors

    컬러는 하나의 일관된 서비스로 보이게 하는 일등공신입니다. 특히 리디를 상징하는 Dodger Blue는 우리를 돋보이게 함과 동시에 강력한 브랜드 아이덴티티를 가집니다.

    더 알아보기

  • Web

    수많은 책 중에서 꼭 맞는 책을 찾기란 쉽지 않습니다. 리디의 웹 서비스는 좋은 책을 탐색하기 쉽게 도와주며 물 흐르듯 편리한 구매 경험을 자랑합니다.

    더 알아보기

  • App

    모름지기 전자책이라면 환경에 구애받지 않고 편하게 읽을 수 있어야 합니다. 리디의 뷰어 애플리케이션은 언제 어디서나 최상의 독서 경험을 제공합니다.

    더 알아보기
